# permutations
Permutations map numbers to themselves.    
So the input of the program has to be for example 3,2,1 for the identity 1,2,3.   

This won't work:    
15,13,2,4   
This works (each number from 1 to 15 contained):   
13,12,11,4,7,15,6,2,8,3,5,1,9,14,10
